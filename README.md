# E-Commerce Sales Data Analysis

## Project Overview
This project analyzes an e-commerce sales dataset using Excel, MySQL, and Power BI to identify sales trends and customer behavior.

## Tools Used
- Excel
- MySQL
- Power BI
- GitHub

## Project Steps

- SQL Analysis
- Power BI Dashboard
- Business Insights

## Dashboard Includes
